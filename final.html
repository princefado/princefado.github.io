Here's my final project! Kerem and I decided to program a paint-by-number game based on Khan Academy's Tic-Tac-Toe lesson. Using the grid object that they created,
as well as the Khan button class, we made this program. The program is very simple; click on one of the three buttons to get started. From there, fill in the squares
with the appropriate color. To switch colors, click on the numbered colors beneath the grid. When you've filled them all in correctly, the game will display a message
which allows you to return to the home screen.
